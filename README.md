📊 Project Overview

The Sales Analysis Dashboard is a Business Intelligence (BI) project developed using Power BI to analyze and visualize sales data. It provides interactive reports and dashboards that help businesses monitor key performance indicators (KPIs), identify sales trends, evaluate product performance, and make data-driven decisions.

The dashboard transforms raw sales data into meaningful insights by using Power Query for data cleaning, SQL for querying (optional), and DAX for advanced calculations.

🎯 Project Objectives
Analyze overall sales performance.
Monitor revenue and profit trends.
Identify top-performing products and categories.
Compare sales across different regions.
Analyze customer purchasing behavior.
Track monthly and yearly business growth.
Enable interactive data exploration using filters and slicers.
